## Heart Disease Prediction using Machine Learning

This project is developed as part of the Machine Learning Internship at CodeAlpha.

The objective of this project is to predict the possibility of heart disease using patient medical data and machine learning classification algorithms.

---

# Project Objective

To build a machine learning model that can predict whether a patient is likely to have heart disease based on medical attributes such as:

- Age
- Gender
- Chest Pain Type
- Cholesterol Level
- Maximum Heart Rate
- Blood Pressure
- Other health-related parameters

---

# Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

---

# Machine Learning Algorithms Used

1. Logistic Regression  
2. Support Vector Machine (SVM)  
3. Random Forest Classifier  

---

# Dataset

Dataset Used: Heart Disease Dataset (UCI Machine Learning Repository)

The dataset contains various medical attributes used to predict heart disease.

---

# Project Workflow

1. Import Dataset  
2. Data Preprocessing  
3. Train-Test Split  
4. Feature Scaling  
5. Model Training  
6. Model Evaluation  
7. Accuracy Comparison  
8. Final Prediction  

---

# Model Evaluation Metrics

The following evaluation metrics were used:

- Accuracy
- Precision
- Recall
- F1-Score

---

# Results

Among all the models used, the Random Forest Classifier achieved the highest accuracy for heart disease prediction.
